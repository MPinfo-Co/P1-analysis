# P1-analysis — MP-BOX 1.0 需求分析

MP-BOX 1.0 的需求分析與參考文件 repo。

## 這個 repo 是做什麼的

存放需求分析過程中產出的文件，以及開發時的參考資料。
不放程式碼，不放設計稿。

## 目錄結構

```
P1-analysis/
└── references/          # 開發參考資料
    ├── diagrams/        # 系統流程圖、架構圖（Mermaid）
    ├── PRD-template.md  # 產品需求文件範本
    ├── TDD-template.md  # 技術設計文件範本
    └── ...              # 其他參考文件
```

## 各資料夾說明

| 資料夾 / 檔案 | 內容 |
|---|---|
| `references/` | 開發參考資料、規範文件、範本 |
| `references/diagrams/` | 系統架構圖、流程圖、ER 圖（Mermaid 格式） |

| Repo | 用途 |
